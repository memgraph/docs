Getting started

How to Connect to Memgraph Cloud#

To query Memgraph from the graphical user interface, please download and install Memgraph Lab.

To query Memgraph from the terminal please use Memgraph Console.

To query Memgraph from your application please use one of the following client libraries:

  • Memgraph C Client
  • Memgraph Python Client
