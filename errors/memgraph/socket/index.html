info

If you are having trouble dealing with an error, please let us know on the Discord.

Memgraph cannot bind to or listen on socket endpoint‚Äã

Make sure that the specified port (the default port Memgraph runs on is 7687) isn't being used by another process or that you haven't already started another Memgraph instance on the same port.

note

If you weren't able to find the error, please submit it through a Support Ticket so we can look into it and get back to you.
