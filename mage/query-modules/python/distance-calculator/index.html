distance_calculator

Abstract#

The distance calculator is a module for calculating distance between two geographic locations. It measures the distance along the surface of the earth. Formulae takes into consideration the radius of the earth. For this algorithm, it is necessary to define an object that has longitude and latitude properties like this:

(location:Location {lat: 44.1194, lng: 15.2314})

Procedures#

single(start, end, metrics)#

Input:#

  • start: Vertex âž¡ Starting point to measure distance. Required to have lng and lat properties.
  • end: Vertex âž¡ Ending point to measure distance. Required to have lng and lat properties.
  • metrics: str âž¡ Can be either "m" or "km". These stand for meters and kilometers respectively.

Output:#

  • distance: double âž¡ The final result obtained by calculating distance (in 'm' or 'km') between the 2 points that each have its latitude and longitude properties.

Usage:#

MATCH (n:Location), (m:Location)CALL distance_calculator.single(m, n, 'km')YIELD distanceRETURN distance;

multiple(start_points, end_points, metrics)#

Input:#

  • start_points: List[Vertex] âž¡ Starting points to measure distance collected in a list. Required to have lng and lat properties. Must be of the same size as end_points.
  • end_points: List[Vertex] âž¡ Ending points to measure distance collected in a list. Required to have lng and lat properties. Must be of the same size as start_points.
  • metrics: str âž¡ Can be either "m" or "km". These stand for metres and kilometres respectively.

Output:#

  • distance: List[double] âž¡ The final result obtained by calculating distance (in meters) between the 2 points who each have its latitude and longitude.

Usage:#

MATCH (n), (m)WITH COLLECT(n) AS location_set1, COLLECT(m) AS location_set2CALL distance_calculator.multiple(location_set1, location_set2, 'km') YIELD distancesRETURN distances;
